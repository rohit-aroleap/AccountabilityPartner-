## Summary
- Dashboard ideas: 112 across 12 categories
- Prompt engineering ideas: 97 across 14 categories
- Top emergent themes:
  1. **Specificity gap** — Dashboard and prompts are both "good enough" but not specific enough. The sidebar needs last-message preview, the prompt needs actual exercise names, analytics needs per-customer breakdowns. Multiple categories expose the same root cause.
  2. **Operator cognitive load at scale** — The current one-customer-at-a-time model works for 10 customers but breaks at 30+. No search, no sort, no priority queue, no batch actions. Many ideas in categories 3, 6, 7 center on reducing this overhead.
  3. **Unclosed feedback loop** — The 👍/👎 system exists but ratings don't automatically improve prompts, there is no pass-rate metric, no A/B comparison. Multiple categories converge on "close the loop so the AI compounds quality over time."
