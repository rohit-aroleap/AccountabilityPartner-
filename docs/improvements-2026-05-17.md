## Summary
- Dashboard ideas: **96** across 12 categories
- Prompt engineering ideas: **98** across 14 categories
- Top emergent themes across both lists:
  1. **Context richness gap** — the data is already there (progressive overload, exercise names, onboarding answers, notes field, monthly habit history) but it is NOT being injected into prompts; fixing this is the highest-leverage prompt improvement cluster.
  2. **Scaling the operator workflow** — the dashboard is built for one operator managing ~10 customers; as the list grows to 50+, the absence of search, filters, bulk actions, and urgency sorting will become the primary pain.
  3. **Quality loop closure** — ratings are collected but there is no closed loop back to the prompt (no version history, no aggregate view, no golden test set); closing this loop is where the most AI quality leverage lives.
